# 西湖群山徒步导览

> 「西湖群山徒步导览」是一个由民间团体自组织的一次社会创新设计尝试👀。其目的在于帮助「徒步登山者」提高在杭进行徒步的体验。第一期的目标范围设定为「老和云起——北高峰线」的徒步体验优化实践。此群为项目工作群兼开放社群，欢迎提问，欢迎批评，也欢迎加入我们一起共创👏👏👏。
